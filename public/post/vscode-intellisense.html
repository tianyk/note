什么时候需要使用 –global 参数

  • 如果安装的包使用 script 标记来引用(如 jQuery )(也就是在浏览器中使用)
  • 这个包是属于环境的一部分(如 node )时
  • 该包没有使用 –global 安装失败时
  • 如果报错了说需要加 global 时加–global
