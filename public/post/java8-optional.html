Java Optional

Optional类是为了避免空指针问题在Java8中新增的一个类。Optional并不是真正避免避免空指针的，他只是用来提醒我们需要注意空指针问题，我们应该先调用它的isPresent()发放去检查一下实际的值是不是空。

函数返回值，我们时常会忘记判断返回值是不是空。如果返回值是Optional<T>类型，这会提醒我们去检查。

注意：如果返回值是Optional<T>类型，我们都会假定这个返回值不可能是null，不然Optional类就毫无意义了。
