数据类型

最新的 ECMAScript 标准定义了 7 种数据类型:

  • 6 种 原始类型:
    • Boolean
    • Null
    • Undefined
    • Number
    • String
    • Symbol (ECMAScript 6 新定义)
  • 和 Object

可以使用typeof操作符判断对象类型
