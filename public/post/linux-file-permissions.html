linux文件权限

Keke的个人网站
2017-03-21

chmod是Linux下设置文件权限的命令，后面的数字表示不同用户或用户组的权限。

一般是三个数字：
第一个数字表示文件所有者的权限
第二个数字表示与文件所有者同属一个用户组的其他用户的权限
第三个数字表示其它用户组的权限。

位 说明
7 文件所有者可读可写可执行
5 与文件所有者同属一个用户组的其他用户可读可执行
5 其它用户组可读可执行

chattr [+/-]i filename

x 权限

  • 文件

    有权限执行 ./

  • 目录

    有权限进入 cd

r 权限

  • 文件

    有权限读 cat

  • 目录

    有权限查看 ls

w 权限

  • 文件

    有权限写 echo hello >

  • 目录

    有权限写入 touch 有权限删除 rm
